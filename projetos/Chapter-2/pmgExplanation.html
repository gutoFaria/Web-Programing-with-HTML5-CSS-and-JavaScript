Método map() em javascript

Conceito

O map faz parte do conjunto de métodos disponíveis na linguagem javascript para a manipulação de objetos do tipo array.Ele opera com par de valores-chave.O método map() é invocado a partir de um array.

Exemplo

var numeros = [1,2,3,4,5];
var quadrados = numeros.map(function(item){
return Math.pow(item,2);
})
